Novo episódio
Adicione uma URL por qualidade. Hospede os vídeos no GitHub Releases (não no repositório).
Exemplo: https://github.com/SEU-USER/SEU-REPO/releases/download/v1.0/ep01-1080p.mp4
Faixa sem URL usa o áudio do próprio vídeo (ex: original japonês).
Faixa com URL usa um arquivo .aac/.mp3 separado (ex: dublagem PT).
Áudio dublado: extraia com FFmpeg e suba no Releases.
Comando: ffmpeg -i video-dub.mkv -vn -c:a aac -b:a 128k dub-pt.aac
Coloque os arquivos .ass na pasta legendas/ do seu repositório.
⚠️ Não use GitHub Releases para .ass!
O navegador bloqueia fetch de Releases por CORS. Use sempre o caminho ./legendas/arquivo.ass.
